    Jelaskan perbedaan antara fungsi (function) dan prosedur (procedure) dalam pemrograman berorientasi objek.
    Mengapa penting menggunakan konstanta (final variables) dalam suatu kelas? Apa keuntungannya?
    Apa yang terjadi jika kita mencoba mengubah nilai suatu konstanta setelah objek dibuat? Berikan contoh.
    Bagaimana cara mendefinisikan konstanta dalam bahasa Java?

    Jawaban

    1. Perbedaan antara fungsi (function) dan prosedur (procedure) dalam pemrograman berorientasi objek

        FUNGSI ;
          1. Mengabalikan nilai ketika dipanggil
          2. digunakan ketika hasil perhitungan atau operasi masih diperlukan oleh kode lain
          3. memiliki tipe return (int, String, boolean,dll)

        PROSEDUR :
        1. tidak mengembalikan ilai (void)
        2. Digunakan untuk menjalankan perintah tanpa perlu mendapatkan nilai kembali
        3. return type nya void

    2. Mengapa penting menggunakan konstanta (final variables) dalam suatu kelas? Apa keuntungannya

        1. Nilainya tidak bisa berubah, sehingga lebih aman dan mencegah bug.
        2. Meningkatkan kejelasan kode, karena menunjukkan bahwa nilai tersebut adalah nilai tetap.
        3. Digunakan untuk konfigurasi atau nilai penting, seperti PI, batas maksimum, atau URL server.
        4. Membantu enkapsulasi, karena membuat data lebih terlindungi.
        5. Lebih efisien, karena JVM bisa melakukan optimasi pada nilai yang bersifat tetap.

    3. Apa yang terjadi jika kita mencoba mengubah nilai suatu konstanta setelah objek dibuat? Berikan contoh.

         EROR,Karena final menjadikan variabel tersebut immutable (tidak dapat diubah)

         CONTOH :
         class Tes {
             final int ANGKA = 10;

             void ubah() {
                 ANGKA = 20; // ERROR: cannot assign a value to final variable ANGKA
             }
         }

            output :Error: cannot assign a value to final variable ANGKA

    4. Bagaimana cara mendefinisikan konstanta dalam bahasa Java?

        1. final
       Untuk membuat variabel tidak dapat diubah

        2.  biasanya juga static
       Supaya nilai konstanta dimiliki oleh kelas, bukan oleh objek

       contoh :
       class Konstanta {
           public static final double PI = 3.14159;
           public static final int MAX_USER = 100;
           public static final String APP_NAME = "MyApplication";
       }

        Nama konstanta ditulis huruf kapital dengan underscore
